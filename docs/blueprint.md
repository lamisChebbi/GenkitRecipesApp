# **App Name**: Minty Recipes

## Core Features:

- Dietary Input: Allow users to input dietary restrictions and preferences (e.g., vegetarian, gluten-free, allergies).
- Recipe Suggestion: Generate personalized recipe ideas based on user's dietary inputs using a generative AI tool.
- Recipe Details: Display detailed ingredient lists, step-by-step instructions, and nutritional information for each recipe suggestion.
- Save Favorite Recipes: Enable users to save their favorite recipes for easy access.

## Style Guidelines:

- Primary color: Fresh mint green (#522ED4) to reflect freshness and health.
- Background color: Clean white (#F0F8FF) for a simple, trendy interface.
- Accent color: Light gray (#D3D3D3) for subtle contrast and a modern feel.
- Body and headline font: 'PT Sans' for a clean and readable aesthetic.
- Use minimalist icons for navigation and recipe actions.
- Employ a clean, grid-based layout for recipe presentation and user interface elements.
- Incorporate subtle transitions and animations when displaying recipes or user interactions.
